To run:

1) Open terminal and move into the UkuleleByEar folder - so if you opened the zip
file and put the folder in Desktop you would use the command:
cd Desktop/UkuleleByEar

2) Enter the following commands:
python3 -m venv venv
. venv/bin/activate
pip install Flask
export FLASK_APP=server.py
flask run

3) Open a browser window and go to http://127.0.0.1:5000/
